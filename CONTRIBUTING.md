# Contributing to Bobo Auth

Thanks for your interest in contributing to Bobo Auth!

## How to Contribute

- Open an issue before starting major work.
- Keep pull requests focused and small.
- Follow the existing coding style.
- Write clear commit messages.

## Development Setup

Documentation for local development will be added once the public API stabilizes.

## Code of Conduct

By participating, you agree to uphold the project's Code of Conduct.
